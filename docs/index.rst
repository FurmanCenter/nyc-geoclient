Python Bindings for NYC Geoclient
=================================

Python bindings for the REST `NYC Geoclient API`_.

.. _NYC Geoclient API: http://developer.cityofnewyork.us/api/geoclient-api-beta

Release v\ |version|.

The NYC Geoclient API provides accurate geocoding for New York City.  It
supports lookups based off of address, BBL, BIN, blockface, intersection, and
place name.  It also provides informative error messages in cases where the
geocoding fails.

These Python bindings make it easier to make requests to the API.
